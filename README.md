# sistema-de-vendas-jpa
Sistema de vendas em Java com JPA e Hibernate para conexão com o banco de dados. <br />
Utilizando:
- Testes unitários;
- Testes de integração;
- TDD;
- JPA;
- Hibernate
- PostgreSQL;
- Mock;
- Generics;
- Annotations;
- Reflections;
- Exceptions;
- Camada de Service e de DAO.
